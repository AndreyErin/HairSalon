REST Api-сервер (Back-End часть для сайта парикмахерской)

Api для сайта(и мобильных клиентских приложений):
	Выдача списка услуг;
	Выдача данных конфигурации (включена ли акции и тд);
	Запись на выбранную услугу в выбранное время;

Api для админского мобильного приложения:
	Аутентификация по средствам Jwt-токена;
	Просмотр рассписания записей на услуги;
	Редактирование услуг;
	Изменение кофигурации сайта (включена ли акции и тд);

Админка на Razor(Admin - Admin123$):
	Редактирование списка сотрудников;
	Выбор дней/времени доступных для записи для конкретного сотрудника;
	Просмотр рассписания записей на услуги;
	Редактирование услуг;
	Изменение кофигурации сайта (включена ли акции и тд);
